About This Website

This website is the official dashboard for the Plant Monitor system.

It allows users to:

  • View live soil moisture readings
  • View live soil temperature readings
  • Track plant health over time
  • Adjust system settings such as the dry threshold
  • Connect directly to the Arduino using USB
  • View real‑time graphs of sensor data
  • Switch between light and dark mode

The website is designed to be simple, modern, and easy to use. It acts as the main control center for the Plant Monitor hardware system.
